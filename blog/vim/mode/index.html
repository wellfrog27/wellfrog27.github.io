模式

作者： 阮一峰

日期： 2015年7月8日

Vim 分成多种模式。按i键进入编辑模式，按esc键退出编辑模式。

  • 命令模式
  • 编辑模式
  • 可视模式

可视模式

  • v - Enter visual mode
  • ctrl+v - Enter visual mode (vertical)
  • V - Enter visual mode (full lines)

在窗口底部输入:set showmode，以后窗口底部就会显示当前状态，比如输入模式是---INSERT--。
