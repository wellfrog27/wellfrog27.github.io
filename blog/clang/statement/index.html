语句

作者： 阮一峰

日期： 2019年1月10日

语句（statement）用来执行某个命令，通常没有返回值。

if 语句

if语句用于判断。

1

if (expression) statement

下面是一个例子。

1
2

if (line_num == MAX_LINES)
  line_num = 0;

多个语句组成的复合语句，必须写在大括号里面。

1
2
3
4

if (line_num == MAX_LINES) {
  line_num = 0;
  page_num++;
}

if语句可以带有else语句。

1
2

if (expression) statement
else statement

下面是一个例子。

1
2
3
4

if (i > j)
  max = i;
else
  max = j;

else可以与另一个if语句连用。

1
2
3
4
5
6
7
8
9

if (expression)
  statement
else if (expression)
  statement
...
else if (expression)
  statement
else
  statement

while 语句

while语句在满足条件时，不断进行循环。

1
2

while (expression)
  statement

下面是一个例子。

1
2

while (i < n)
  i = i + 2;

只要条件为真，就会无限循环。下面是一种常见的写法。

1
2
3

while (1) {
  // ...
}

do 语句

do语句先执行一次循环体，然后再判断是否还要执行。

1
2

do statement
while (表达式);

它与while语句的区别是，不管怎样，循环体至少会执行一次。

1
2
3

do {
  --i;
} while (i > 0);

for 语句

for语句通常用于精确控制循环次数。

1
2

for (表达式1; 表达式2; 表达式)
  语句

下面是一个例子。

1
2

for (i = 10; i > 0; i--)
  printf("%d\n", i):

循环变量可以在for的第一个表达式里面声明。

1

for (int i = 0; i < n; i++)
