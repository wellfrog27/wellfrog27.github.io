运算符

作者： 阮一峰

日期： 2019年2月10日

C 语言的运算符非常多，一共有 50 多种。

算术运算符

  • +：一元的正值运算符
  • -：一元的负值运算符
  • +：加法运算符
  • -：减法运算符
  • *：乘法运算符
  • /：除法运算符
  • %：余值运算符

赋值运算符

  • = 赋值运算符
  • += 加法赋值运算符
  • -= 减法赋值运算符
  • *= 乘法赋值运算符
  • /= 除法赋值运算符
  • %/ 余值赋值运算符

自增运算符和自减运算符

  • ++：自增运算符
  • --：自减运算符

关系运算符

  • > 大于运算符
  • < 小于运算符
  • >= 大于等于运算符
  • <= 小于等于运算符

判断运算符

  • == 等于运算符
  • != 不等于运算符

逻辑运算符

  • ! 非运算符
  • && 与运算符
  • || 或运算符
